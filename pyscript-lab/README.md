# pyscript-lab

Running python scripts in the browser

## Creating a project

Small Svelte app demonstrating Python<->JS Interoperability with [PyScript](https://pyscript.net)

Run with `npm install` (or `pnpm install` or `yarn`), start a development server:

```bash
npm run dev

# or start the server and open the app in a new browser tab
npm run dev -- --open
```
